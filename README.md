# Medicared

A Modern healthcare management system built with React, TypeScript, Vite, Tailwind CSS, and Supabase.

## Features

* Patient Management
* Doctor Portal
* Appointment Scheduling
* Pharmacy Management
* Laboratory Management
* Billing & Reporting
* Authentication & Security
* Role-Based Access Control
* Supabase Integration

## Tech Stack

* React
* TypeScript
* Vite
* Tailwind CSS
* Supabase

## Prerequisites

Before running the project, ensure you have:

* Node.js 18+
* npm
* Supabase project

## Installation

Clone the repository:

```bash
git clone https://github.com/GRT-Group/medicared-main.git
```

Navigate into the project:

```bash
cd medicared-main
```

Install dependencies:

```bash
npm install
```

Create an environment file:

```bash
cp .env.example .env
```

Update the values inside `.env`.

Start the development server:

```bash
npm run dev
```

The application will be available at:

```text
http://localhost:5173
```

## Build for Production

```bash
npm run build
```

## Run Tests

```bash
npm run test
```

## Environment Variables

See `.env.example` for required variables.
